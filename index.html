TypeScript

TypeScript is a strongly typed programming language that builds on JavaScript, giving you better tooling at any scale.

Benefits Drawbacks
Strong static typing Encreased complexity in codebase
Early error detection Additional build step for compilation
Improved code maintainability Larger file size of compiled code
Strict type checking Potential performance overhead
Easy integration with JavaScript Requires re-familiarization with JavaScript syntax and concepts

There are 6 basic, primitive types to remember:

  1. number
  2. string
  3. boolean
  4. symbol
  5. null
  6. undefined

Never

Never type, according to the pure definition, represents the value that never occurs.

Unknown

Unknown is a relatively new addition to TS types collection - it was introduced in v3.0. It's meant to be a type-safe alternative to any type.

            let myUnknown: unknown = "str"
            let myAny: any = "str"
            let myUnknownVariable2: unknown = myUnknownVariable
            myUnknownVariable2 = myAnyVariable
            let myStringVariable: string = "str";
            myStringVariable = myAnyVariable;
            myStringVariable = myUnknownVariable // error

Composed types

Unions
            let myUnionVariable: string | number = "str"; 
            myUnionVariable = 10; 
             myUnionVariable = false; // error
Literals
            let myStringLiteral: "str" = "str"; 
            let myNumberLiteral: 10 = 10;
            myStringLiteral = "string"; // error
            myNumberLiteral = 55; // error
Intersection types

Intersection types are closely related to union types. While union types function like logical or, intersection types function like logical and. Thus, you can create them using the and sign (&).

            const myIntersectionVariable: {str: string} & {num: number} = { 
              str : "str", num: 10
            };

Enums

They are used to simply provide more friendly names to numeric values.

            enum Color {Red, Green, Blue};
            Color.Red // 0 
            Color.Blue // 2 
            Color[1] // "Green" 
            Color[2] // "Blue"

There are three types of enums:

  • Numeric enum
  • String enum
  • Heterogeneous enum

Functions

  1. In TypeScript, functions can be of two types: named and anonymous.
  2. TypeScript provides the option to add default values to parameters.
  3. TypeScript has an optional parameter functionality. The parameters that may or may not receive a value can be appended with a '?' to mark them as optional.
            function Greet(greeting: string, name?: string ) : string {
              return greeting + ' ' + name + '!';
            }
            let greeting = function() {
              console.log("Hello TypeScript!");
            }
